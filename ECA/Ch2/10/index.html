Determine which of the following statemets are true

  • (a) int⁡(cl⁡(A))=int⁡(A)\operatorname{int}(\operatorname{cl}(A)) = \operatorname{int}(A)int(cl(A))=int(A)
  • (b) cl⁡(A)∩A=A\operatorname{cl}(A)\cap A = Acl(A)∩A=A
  • (c) cl⁡(int⁡(A))=A\operatorname{cl}(\operatorname{int}(A)) = Acl(int(A))=A
  • (d) bd⁡(cl⁡(A))=bd⁡(A)\operatorname{bd}(\operatorname{cl}(A)) = \operatorname{bd}(A)bd(cl(A))=bd(A)
  • (e) If AAA is open, then bd⁡(A)⊂M∖A\operatorname{bd}(A)\subset M\setminus Abd(A)⊂M∖A

  • (b) True

    ⊆\subseteq⊆ is trivial and ⊇\supseteq⊇ is done by definition.

  • (d) False.

    Let A=QA = \mathbb{Q}A=Q, then bd⁡(cl⁡(A))=bd⁡(R)=∅\operatorname{bd}(\operatorname{cl}(A)) = \operatorname{bd}(\mathbb{R}) = \emptysetbd(cl(A))=bd(R)=∅ but bd⁡(Q)=R\operatorname{bd}(\mathbb{Q}) = \mathbb{R}bd(Q)=R.
