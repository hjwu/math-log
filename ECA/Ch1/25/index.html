題目​

We say that P≤QP\leq QP≤Q if for each x∈Px\in Px∈P, there is a y∈Qy\in Qy∈Q with x≤yx\leq yx≤y

  • (a) If P≤QP\leq QP≤Q, then show that sup⁡(P)≤sup⁡(Q)\sup(P)\leq\sup(Q)sup(P)≤sup(Q).
  • (b) Is it true that inf⁡(P)≤inf⁡(Q)\inf(P)\leq \inf(Q)inf(P)≤inf(Q) ?
  • (c) If P≤Q,Q≤PP\leq Q, Q\leq PP≤Q,Q≤P, does P=QP=QP=Q ?

解答​

  • (a) If x∈Px\in Px∈P, there is a y∈Qy\in Qy∈Q, s.t. x<y≤sup⁡(Q)x < y \leq \sup(Q)x<y≤sup(Q). Then sup⁡(Q)\sup(Q)sup(Q) is an upper bound of PPP. Hence sup⁡(P)≤sup⁡(Q).\sup(P)\leq\sup(Q).sup(P)≤sup(Q).
  • (b) False. Choose P={ 1 }P=\lbrace\ 1\ \rbraceP={ 1 } and Q={ 0,1 }Q=\lbrace\ 0, 1\ \rbraceQ={ 0,1 }, then inf⁡(P)=1>inf⁡(Q)=0.\inf(P) = 1 > \inf(Q) = 0.inf(P)=1>inf(Q)=0.
  • (c) False. Use the counterexample of (b).
