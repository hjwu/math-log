題目​

For nonempty sets A,B⊂RA,B\subset\mathbb{R}A,B⊂R, determine which of the follwing statement are true. Prove the true statements and give a counterexample for those that are false

  • (a) sup⁡(A∩B)≤inf⁡{ sup⁡(A),sup⁡(B) }\sup(A\cap B)\leq \inf\lbrace\ \sup(A),\sup(B)\ \rbracesup(A∩B)≤inf{ sup(A),sup(B) }
  • (b) sup⁡(A∩B)=inf⁡{ sup⁡(A),sup⁡(B) }\sup(A\cap B) = \inf\lbrace\ \sup(A),\sup(B)\ \rbracesup(A∩B)=inf{ sup(A),sup(B) }
  • (c) sup⁡(A∪B)≥sup⁡{ sup⁡(A),sup⁡(B) }\sup(A\cup B) \geq \sup\lbrace\ \sup(A),\sup(B)\ \rbracesup(A∪B)≥sup{ sup(A),sup(B) }
  • (d) sup⁡(A∪B)=sup⁡{ sup⁡(A),sup⁡(B) }\sup(A\cup B) = \sup\lbrace\ \sup(A),\sup(B)\ \rbracesup(A∪B)=sup{ sup(A),sup(B) }

解答​

  • (a) True

    For all x∈A∩Bx\in A\cap Bx∈A∩B, x≤sup⁡(A)x\leq \sup(A)x≤sup(A) and sup⁡(B)\sup(B)sup(B), then x≤inf⁡{ sup⁡(A),sup⁡(B) }.x\leq \inf\lbrace\ \sup(A),\sup(B)\ \rbrace.x≤inf{ sup(A),sup(B) }. So

    sup⁡(A∩B)≤inf⁡{ sup⁡(A),sup⁡(B) }\sup(A\cap B)\leq \inf\lbrace\ \sup(A),\sup(B)\ \rbracesup(A∩B)≤inf{ sup(A),sup(B) }

  • (b) False

    Let A=(0,1)A = (0,1)A=(0,1), B=(1,2)B = (1,2)B=(1,2), then

    sup⁡(A∩B)=sup⁡(∅)=−∞<1=inf⁡{ sup⁡(A)=1,sup⁡(B)=2 }.\sup(A\cap B)=\sup(\emptyset) = -\infty < 1 = \inf\lbrace\ \sup(A)=1,\sup(B)=2\ \rbrace.sup(A∩B)=sup(∅)=−∞<1=inf{ sup(A)=1,sup(B)=2 }.

  • (c) True

    It's trivial that sup⁡(A)\sup(A)sup(A) and sup⁡(B)≤sup⁡(A∪B)\sup(B)\leq \sup(A\cup B)sup(B)≤sup(A∪B). Then

    sup⁡(A∪B)≥sup⁡{ sup⁡(A),sup⁡(B) }\sup(A\cup B) \geq \sup\lbrace\ \sup(A),\sup(B)\ \rbracesup(A∪B)≥sup{ sup(A),sup(B) }

  • (d) True

    For all x∈A∪Bx\in A\cup Bx∈A∪B, x≤sup⁡(A)x\leq \sup(A)x≤sup(A) or x≤sup⁡(B)x\leq \sup(B)x≤sup(B), then sup⁡{sup⁡(A),sup⁡(B)}\sup\lbrace\sup(A),\sup(B)\rbracesup{sup(A),sup(B)} is an upper bound of xxx, i.e.

    sup⁡(A∪B)≤sup⁡{ sup⁡(A),sup⁡(B) }\sup(A\cup B) \leq \sup\lbrace\ \sup(A),\sup(B)\ \rbracesup(A∪B)≤sup{ sup(A),sup(B) }

    by (c) we have the conclusion

    sup⁡(A∪B)=sup⁡{ sup⁡(A),sup⁡(B) }.\sup(A\cup B) = \sup\lbrace\ \sup(A),\sup(B)\ \rbrace.sup(A∪B)=sup{ sup(A),sup(B) }.
