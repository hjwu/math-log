題目​

Let nnn and kkk be integers such that n≥2n\geq 2n≥2 and 1≤k≤n.1\leq k\leq n.1≤k≤n. Define

dk(n)=∣(nk)−(nk−1)∣d_k(n)=\bigg\lvert{n\choose k}-{n\choose k-1}\bigg\rvertdk​(n)=∣∣​(kn​)−(k−1n​)∣∣​

and

dmin⁡(n)=min⁡{dk(n)∣1≤k≤n}.d_{\min}(n)=\min\{d_k(n)\mid 1\leq k\leq n\}.dmin​(n)=min{dk​(n)∣1≤k≤n}.

Prove that for nnn different from 444 and 666, dk(n)=n−1d_k(n) = n-1dk​(n)=n−1 if and only if k=1k=1k=1 or k=nk=nk=n.

解答(法1)​

Only do   ⟹  \implies⟹ part, the other is easy.

Claim <1>​

If nnn is even, say n=2kn=2kn=2k, and n>6n>6n>6, then dk(n)>1d_{k}(n)>1dk​(n)>1.

Proof:

If not, then there exist some kkk such that

(2k)!k!k!−(2k)!(k+1)!(k−1)!=1⇒(2k)!k!(k−1)!=k(k+1)⇒(2k)!=k!(k+1)!⇒∏i=1k(k+i)=∏i=1k(1+i)\begin{aligned} &\quad &\frac{(2k)!}{k!k!} - \frac{(2k)!}{(k+1)!(k-1)!} &= 1\\ &\Rightarrow &\frac{(2k)!}{k!(k-1)!} &= k(k+1)\\ &\Rightarrow &(2k)! &= k!(k+1)!\\ &\Rightarrow &\prod\limits_{i=1}^{k}(k+i) &= \prod\limits_{i=1}^{k}(1+i) \end{aligned}​⇒⇒⇒​k!k!(2k)!​−(k+1)!(k−1)!(2k)!​k!(k−1)!(2k)!​(2k)!i=1∏k​(k+i)​=1=k(k+1)=k!(k+1)!=i=1∏k​(1+i)​

, but k>1⇒ ⁣⇐.k > 1 \Rightarrow\!\Leftarrow.k>1⇒⇐.

Claim <2>​

For n>6n>6n>6, dk(n)>n−1d_k(n) > n-1dk​(n)>n−1 where 1<k≤⌊n−12⌋1 < k \leq \lfloor\frac{n-1}{2}\rfloor1<k≤⌊2n−1​⌋

Proof:

Use induction on nnn

  1. n=7n=7n=7, easy
  2. Suppose nnn holds, then
dk(n+1)=(n+1k)−(n+1k−1)=((nk)+(nk−1))−((nk−1)+(nk−2))=((nk)−(nk−1))+((nk−1)−(nk−2))=dk(n)+dk−1(n)\begin{aligned} d_k(n+1) &= {n+1 \choose k} - {n+1 \choose k-1}\\ &= \bigg( {n\choose k} + {n\choose k-1} \bigg) - \bigg( {n\choose k-1} + {n\choose k-2} \bigg)\\ &= \bigg( {n\choose k} - {n\choose k-1} \bigg) + \bigg( {n\choose k-1} - {n\choose k-2} \bigg)\\ &= d_{k}(n) + d_{k-1}(n) \end{aligned}dk​(n+1)​=(kn+1​)−(k−1n+1​)=((kn​)+(k−1n​))−((k−1n​)+(k−2n​))=((kn​)−(k−1n​))+((k−1n​)−(k−2n​))=dk​(n)+dk−1​(n)​

If 1<k≤⌊n−12⌋1 < k \leq \lfloor\frac{n-1}{2}\rfloor1<k≤⌊2n−1​⌋, by induction we have

dk(n+1)=dk(n)+dk−1(n)≥2(n−1)=n+(n−2)>nd_{k}(n+1) = d_{k}(n) + d_{k-1}(n) \geq 2(n-1) = n+(n-2) > ndk​(n+1)=dk​(n)+dk−1​(n)≥2(n−1)=n+(n−2)>n

, the equality part is for k=2k=2k=2.

Otherwise, ⌊n−12⌋<k≤⌊n2⌋\lfloor\frac{n-1}{2}\rfloor < k \leq\lfloor\frac{n}{2}\rfloor⌊2n−1​⌋<k≤⌊2n​⌋, that means n=2kn=2kn=2k. Hence by induction and claim <1>, we have

dk(n+1)=dk(n)+dk−1(n)>1+n−1=n.d_{k}(n+1) = d_{k}(n) + d_{k-1}(n) > 1 + n-1 = n.dk​(n+1)=dk​(n)+dk−1​(n)>1+n−1=n.

Claim <3>​

If nnn is even, say n=2kn=2kn=2k, and n>6n>6n>6, then dk(n)≠n−1d_{k}(n)\neq n-1dk​(n)=n−1.

Proof:

If not, then there exist some k>3k>3k>3 such that dk(n)=n−1d_{k}(n)=n-1dk​(n)=n−1. Then

(2k)!k!k!−(2k)!(k+1)!(k−1)!=2k−1⇒(2k)!k!(k−1)!=(2k−1)k(k+1)⇒2(2k−2)!=(k+1)!(k−1)!⇒∏i=0k−2(k+i)=∏i=0k−2(3+i)\begin{aligned} &\quad&\frac{(2k)!}{k!k!} - \frac{(2k)!}{(k+1)!(k-1)!} &= 2k-1\\ &\Rightarrow &\frac{(2k)!}{k!(k-1)!} &= (2k-1)k(k+1)\\ &\Rightarrow &2(2k-2)! &= (k+1)!(k-1)!\\ &\Rightarrow &\prod\limits_{i=0}^{k-2}(k+i) &= \prod\limits_{i=0}^{k-2}(3+i) \end{aligned}​⇒⇒⇒​k!k!(2k)!​−(k+1)!(k−1)!(2k)!​k!(k−1)!(2k)!​2(2k−2)!i=0∏k−2​(k+i)​=2k−1=(2k−1)k(k+1)=(k+1)!(k−1)!=i=0∏k−2​(3+i)​

, but k>3⇒ ⁣⇐.k > 3 \Rightarrow\!\Leftarrow.k>3⇒⇐.

Finally​

By claim <2>, it's clearly that dk(n)>n−1d_{k}(n) > n-1dk​(n)>n−1, whenever n>6n > 6n>6, for ⌈n+12⌉≤k<n.\lceil\frac{n+1}{2}\rceil\leq k < n.⌈2n+1​⌉≤k<n.

Hence by claim <2> and claim <3> we have dk(n)≠n−1d_{k}(n) \neq n-1dk​(n)=n−1, whenever n>6n > 6n>6, and when n = 2,3,52,3,52,3,5, it's easy to check that dk(n)=n−1d_k(n) = n-1dk​(n)=n−1 only if k=1k=1k=1 or k=nk=nk=n.

解答(法2)​

Suppose there exist kkk is neither 1 nor nnn s.t. dk(n)=n−1d_k(n)=n-1dk​(n)=n−1, let 2≤k≤n22\leq k \leq \frac{n}{2}2≤k≤2n​, then we can rewrite

dk(n)=n!k!(n−k)!−n!(k−1)!(n−k+1)!=n−1d_k(n)=\frac{n!}{k!(n-k)!} - \frac{n!}{(k-1)!(n-k+1)!} = n-1dk​(n)=k!(n−k)!n!​−(k−1)!(n−k+1)!n!​=n−1

as

(n−2k+1)×n×(n−2)!=(k!)×(n−k+1)!(⋆)\tag{⋆}(n-2k+1)\times n\times(n-2)! = (k!)\times(n-k+1)!(n−2k+1)×n×(n−2)!=(k!)×(n−k+1)!(⋆)

Case : k = 2​

Then the equation (⋆)(\star)(⋆) becomes (n−3)n(n−2)!=2!(n−1)!(n-3)n(n-2)! = 2!(n-1)!(n−3)n(n−2)!=2!(n−1)!, hence we have n2−5n+2=0n^2 - 5n + 2 = 0n2−5n+2=0. However the root nnn is not an integer. ⇒ ⁣⇐\Rightarrow\!\Leftarrow⇒⇐

Case : k = 3​

Then the equation (⋆)(\star)(⋆) becomes (n−5)n(n−2)!=3!(n−2)!(n-5)n(n-2)! = 3!(n-2)!(n−5)n(n−2)!=3!(n−2)!, hence we have n=6n=6n=6 is the only one positive solution. However n≠6.⇒ ⁣⇐n\neq 6. \Rightarrow\!\Leftarrown=6.⇒⇐

Case : k > 3​

Then the equation (⋆)(\star)(⋆) becomes

(n−2k+1)×n×∏i=0k−4(n−k+2+i)=2k×∏i=0k−4(3+i)(n-2k+1)\times n\times\prod\limits_{i=0}^{k-4}(n-k+2+i) = 2k\times\prod\limits_{i=0}^{k-4}(3+i)(n−2k+1)×n×i=0∏k−4​(n−k+2+i)=2k×i=0∏k−4​(3+i)

Since n≥2kn\geq 2kn≥2k, then n−2k+1≥1,  n−k+2≥k+2>3n-2k+1\geq1,\; n-k+2\geq k+2>3n−2k+1≥1,n−k+2≥k+2>3. So we have contradiction:

(n−2k+1)×n×∏i=0k−4(n−k+2+i)>1×2k×∏i=0k−4(3+i)(n-2k+1)\times n\times \prod\limits_{i=0}^{k-4}(n-k+2+i) > 1\times 2k \times \prod\limits_{i=0}^{k-4}(3+i)(n−2k+1)×n×i=0∏k−4​(n−k+2+i)>1×2k×i=0∏k−4​(3+i)

By all case above and symmetry we know it's true for n2+1≤k≤n−1\frac{n}{2}+1 \leq k \leq n-12n​+1≤k≤n−1.

And last case is that k=n+12k = \frac{n+1}{2}k=2n+1​. Since kkk is integer, it means nnn is odd, and dk(n)=0d_k(n) = 0dk​(n)=0 by (b).

Hence nnn different from 444 and 666, dk(n)=n−1d_k(n) = n-1dk​(n)=n−1 if and only if k=1k =1k=1 or k=nk = nk=n.
