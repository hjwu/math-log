Let nnn and kkk be integers such that n≥2n\geq 2n≥2 and 1≤k≤n1\leq k\leq n1≤k≤n. Define

dk(n)=∣(nk)−(nk−1)∣d_k(n)=\bigg\lvert{n\choose k}-{n\choose k-1}\bigg\rvertdk​(n)=∣∣​(kn​)−(k−1n​)∣∣​

dmin⁡(n)=min⁡{dk(n)∣1≤k≤n}.d_{\min}(n)=\min\{d_k(n)\mid 1\leq k\leq n\}.dmin​(n)=min{dk​(n)∣1≤k≤n}.

Prove that dmin⁡(n)=0d_{\min}(n)=0dmin​(n)=0 if and only if nnn is odd.

  ⟹  \implies⟹​

If dmin⁡(n)=0d_{\min}(n) = 0dmin​(n)=0 then dk(n)=0d_k(n) = 0dk​(n)=0 for some kkk. Then

(nk)=(nk−1)⇒(n−k)!k!=(n−k+1)!(k−1)!⇒k=(n−k+1)⇒n=2k−1\begin{aligned} &\quad &{n\choose k} &= {n\choose k-1}\\ &\Rightarrow &(n-k)!k! &= (n-k+1)!(k-1)!\\ &\Rightarrow &k &= (n-k+1)\\ &\Rightarrow &n &= 2k-1 \end{aligned}​⇒⇒⇒​(kn​)(n−k)!k!kn​=(k−1n​)=(n−k+1)!(k−1)!=(n−k+1)=2k−1​

, hence nnn is odd.

If nnn is odd, pick k=(n+1)/2k = (n+1)/2k=(n+1)/2, then

0≤dmin⁡(n)≤dk(n)=∣(nk)−(nk−1)∣=0.0 \leq d_{\min}(n) \leq d_k(n) = \bigg\lvert {n\choose k}-{n\choose k-1} \bigg\rvert = 0.0≤dmin​(n)≤dk​(n)=∣∣​(kn​)−(k−1n​)∣∣​=0.
