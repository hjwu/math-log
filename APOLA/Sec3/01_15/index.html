01_15

題目​

If M\bm{M}M is a matrix such that MA=AM\bm{MA}=\bm{AM}MA=AM, for all matrices A\bm{A}A, show that MA\bm{MA}MA must be scalar matrix.

info

All the problem in sec. 3 are build on n-by-n matrix

提示​

Choose Ei,j=A\bm{E}_{i,j} = \bm{A}Ei,j​=A.
