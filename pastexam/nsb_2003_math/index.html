2003 國安局三等考數理組 - 初等數論

題目​

  1. 設 ppp 是質數， a,ba, ba,b 是整數，如果 ap−bpp\frac{a^p-b^p}{p}pap−bp​ 是整數，證明 ap−bpp2\frac{a^p-b^p}{p^2}p2ap−bp​ 也是整數。
  2. 設 x, y, z 是整數。如果 x3+y3=z3x^3 + y^3 = z^3x3+y3=z3 證明這三個數中必有一個是 7 的倍數。
  3. 證明 1001!1001!1001! 以 249249249 個零為結尾。
  4. 證明對所有整數 x,yx, yx,y, x2+y2−34\frac{x^2+y^2-3}{4}4x2+y2−3​ 不是整數。

解答​

Q1​

如果 a=ba=ba=b，則結束。否則不失一般性，令 a=b+c>ba=b+c>ba=b+c>b，其中 ccc 為正整數。又已知(易證，略過)

If ppp is prime, then (pk){p \choose k}(kp​) is divisible by ppp if and only if integer 0<k<p0<k<p0<k<p

可令 (pk)=pvk{p \choose k}=pv_k(kp​)=pvk​。所以

由上可知 cpp\frac{c^p}{p}pcp​ 也是整數，又 p≥2p\geq 2p≥2 是質數，所以 ccc 是 ppp 的倍數。可令 c=pvc=pvc=pv，所以

ap−bpp2=∑k=1p−1vkckbp−kp+cpp2=c∑k=1p−1vkck−1bp−kp+vpppp2=v∑k=1p−1vkck−1bp−k+vppp−2\begin{align*} \frac{a^p-b^p}{p^2} &= \frac{\sum\limits_{k=1}^{p-1}v_kc^{k}b^{p-k}}{p} + \frac{c^p}{p^2}\\ &= \frac{c\sum\limits_{k=1}^{p-1}v_kc^{k-1}b^{p-k}}{p} + \frac{v^pp^p}{p^2}\\ &= v\sum\limits_{k=1}^{p-1}v_kc^{k-1}b^{p-k} + v^pp^{p-2} \end{align*}p2ap−bp​​=pk=1∑p−1​vk​ckbp−k​+p2cp​=pck=1∑p−1​vk​ck−1bp−k​+p2vppp​=vk=1∑p−1​vk​ck−1bp−k+vppp−2​

Q2​

如果整數 nnn 不是 777 的倍數，則 n3≡±1mod  7n^3\equiv \pm1 \mod 7n3≡±1mod7。

假設 x,yx, yx,y 都不是 777 的倍數，則 x3+y3≡±2,0mod  7x^3 + y^3\equiv \pm2, 0 \mod 7x3+y3≡±2,0mod7。

然而 z3≡±1,0mod  7z^3\equiv \pm1, 0 \mod 7z3≡±1,0mod7，所以 z3≡0mod  7z^3\equiv 0 \mod 7z3≡0mod7，即 zzz 是 777 的倍數

For every integer nnn, then n2≡1,0mod  4n^2\equiv 1, 0 \mod 4n2≡1,0mod4.
